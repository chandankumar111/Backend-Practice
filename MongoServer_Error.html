How To FIx MongoDB Server Connection Error

1. download MongoShell from official website

2. Extract the mongoshell folder and rename it with mongosh and paste it on folder C:\Program Files

3. go to mongosh folder and find bin folder and copy the path of bin folder and set it into the system varible

4. create a folder name of data on C:// drive

5. create a folder name with db inside the data folder

6. reopen the cmd and type mongod to start mongodb server

7. at last to use mongoshell open new cmd and type mongosh to start mongoshell to use mongodb
